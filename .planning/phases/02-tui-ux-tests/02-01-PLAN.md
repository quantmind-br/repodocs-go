Follow the exact pattern from `ollama_test.go` but adapted for LM Studio's OpenAI-compatible wire format. Include these test functions:

1. **TestNewLMStudioProvider** -- constructor test:
   - Create provider with `ProviderConfig{BaseURL: "http://localhost:1234/v1", Model: "test-model", MaxTokens: 1000, Temperature: 0.7}`
   - Assert `provider.Name() == "lmstudio"`, no error, not nil

2. **TestLMStudioProvider_Complete_Success** (TEST-01):
   - httptest server handler verifies: `r.Method == "POST"`, `r.URL.Path == "/chat/completions"`, `r.Header.Get("Content-Type") == "application/json"`, `r.Header.Get("Authorization") == ""` (no auth)
   - Decode request body with `decodeJSON` into `map[string]interface{}`, assert `reqBody["model"] == "lmstudio-model"`
   - Server responds with: `{"id":"test-1","model":"lmstudio-model","choices":[{"message":{"role":"assistant","content":"Test response"},"finish_reason":"stop"}],"usage":{"prompt_tokens":10,"completion_tokens":5,"total_tokens":15}}`
   - Assert response: `Content == "Test response"`, `Model == "lmstudio-model"`, `FinishReason == "stop"`, `Usage.PromptTokens == 10`, `Usage.CompletionTokens == 5`, `Usage.TotalTokens == 15`

3. **TestLMStudioProvider_Complete_WithAPIKey** (TEST-02 part A):
   - Capture `r.Header.Get("Authorization")` in handler
   - Create provider with `APIKey: "my-secret-key"`
   - Assert captured auth == `"Bearer my-secret-key"`

4. **TestLMStudioProvider_Complete_WithoutAPIKey** (TEST-02 part B):
   - Same pattern but `APIKey: ""` (omitted)
   - Assert `r.Header.Get("Authorization") == ""` (empty)

5. **TestLMStudioProvider_Complete_APIError** -- 500 error:
   - Server returns 500 with `{"error":{"message":"internal server error","type":"server_error"}}`
   - Assert `domain.LLMError` with `Provider == "lmstudio"`, `StatusCode == 500`

6. **TestLMStudioProvider_Complete_RateLimit** -- 429 error:
   - Server returns 429 with `{"error":{"message":"rate limit exceeded"}}`
   - Assert `domain.LLMError` with `Provider == "lmstudio"`, `StatusCode == 429`, `ErrorIs(llmErr, domain.ErrLLMRateLimited)`

7. **TestLMStudioProvider_Complete_AuthError** -- 401 error:
   - Server returns 401 with `{"error":{"message":"invalid api key"}}`
   - Assert `domain.LLMError` with `Provider == "lmstudio"`, `StatusCode == 401`, `ErrorIs(llmErr, domain.ErrLLMAuthFailed)`

8. **TestLMStudioProvider_Complete_EmptyChoices** -- no choices:
   - Server returns 200 with `{"id":"test","model":"m","choices":[],"usage":{}}`
   - Assert error, `domain.LLMError` with `Provider == "lmstudio"`, `Message contains "no choices"`

9. **TestLMStudioProvider_Complete_ConnectionRefused** (CONF-04):
   - Create provider with `BaseURL: "http://localhost:19999"` (nothing listening), `http.Client{Timeout: 1 * time.Second}`
   - Assert error is `*domain.LLMError`, `llmErr.Provider == "lmstudio"`, `strings.Contains(llmErr.Message, "request failed")`
   - This proves CONF-04: connection-refused surfaces as a clear LLMError with provider name and address

10. **TestLMStudioProvider_Complete_ContextCancellation**:
    - httptest server with `time.Sleep(100ms)`, create cancelled context, assert error

11. **TestLMStudioProvider_Complete_InvalidJSONResponse**:
    - Server returns 200 with `{invalid json`
    - Assert error contains "failed to parse response"

12. **TestLMStudioProvider_Complete_NoModelLoaded503**:
    - Server returns 503 with plain text `"no model is currently loaded"`
    - Assert `domain.LLMError` with `Provider == "lmstudio"`, `Message contains "no model is loaded"`

13. **TestLMStudioProvider_Close** -- close returns nil

14. **TestLMStudioProvider_Complete_WithSystemMessage**:
    - Send system + user messages, capture request body, verify messages array has 2 entries with correct roles

Imports needed: `context`, `net/http`, `net/http/httptest`, `testing`, `time`, `github.com/stretchr/testify/assert`, `github.com/stretchr/testify/require`, `github.com/quantmind-br/repodocs/internal/domain`
