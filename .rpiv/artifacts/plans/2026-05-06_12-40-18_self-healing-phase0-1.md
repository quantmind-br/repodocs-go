# Self-Healing Pipeline Phase 0+1 Implementation Plan

## Overview

Build the Phase 0+1 foundation for a self-healing extraction pipeline: strategies return structured extraction outcomes, and the orchestrator validates those outcomes before reporting success. This plan deliberately stops before fallback execution, probes, recipe cache, or UX trees; Phase 1 converts silent 0-document success into a user-facing outcome error with actionable diagnostics.

## Requirements

- Add `domain.StrategyResult` telemetry with discovered/attempted/written/skipped/failed counters, diagnostics, bytes, and duration.
- Change both strategy interfaces (`internal/strategies.Strategy` and `internal/domain.Strategy`) so `Execute` returns `(*domain.StrategyResult, error)`.
- Preserve current extraction behavior while collecting telemetry; Phase 0 must be a mechanical, compiling migration.
- Instrument all concrete strategies enough that Phase 1 validator does not falsely fail successful strategies.
- Add a pure `internal/recovery` validator that flags empty-success outcomes and propagates transient errors.
- Integrate validation into `Orchestrator.Run` after strategy execution and before success logging.
- Preserve explicit dry-run semantics: dry-run validates discovery/attempts, not physical writes.
- Treat skipped existing/synced documents as acceptable output (`DocsWritten + DocsSkipped`).
- Add tests for result counters, validator verdicts, orchestrator empty-output errors, and rust-book-style filter-zeroed diagnostics.

## Current State Analysis

The current pipeline selects one strategy and treats `nil` error as success. Strategies do not report structured output, so the orchestrator cannot distinguish a successful extraction from a no-op. The rust-book case arises because sitemap discovery can switch from crawler to sitemap, then a path filter can reduce the sitemap URLs to zero while `SitemapStrategy.Execute` still returns `nil`.

### Key Discoveries

- `internal/strategies/strategy.go:25-31` defines the strategy contract as `Execute(... ) error`, so output telemetry cannot pass through the orchestrator.
- `internal/domain/interfaces.go:10-17` defines a second `Strategy` interface; user chose to update both interfaces to avoid drift and keep generated mocks aligned.
- `internal/app/orchestrator.go:122-241` detects one strategy, optionally switches to sitemap, executes once, and only checks the error.
- `internal/app/orchestrator.go:155-166` performs sitemap discovery before crawler execution; this is where the rust-book case switches strategy.
- `internal/strategies/sitemap.go:107-113` logs filtered sitemap count and then processes zero URLs without returning an error.
- `internal/strategies/sitemap.go:118-166` processes sitemap indexes and only warns when zero nested URLs are processed.
- `internal/strategies/crawler.go:39-61` already tracks `processedCount`; that counter can seed result telemetry but does not represent successful writes.
- `internal/strategies/crawler.go:210-216` skips unchanged sync pages, and `internal/strategies/sitemap.go:234-236` skips existing files when force is false; these must count as acceptable skips.
- `internal/domain/errors.go:92-112` has the existing pure retryability classifier; `domain.IsTransient` should mirror this style.
- `internal/app/orchestrator.go:35` and `internal/app/orchestrator.go:106-119` provide strategy factory injection, which should be preserved for validator integration tests.
- GitNexus reports CRITICAL blast radius for `internal/strategies/strategy.go:25` (42 dependents), `internal/app/orchestrator.go:122` (59 dependents), `SitemapStrategy.Execute` (34 dependents), and `CrawlerStrategy.Execute` (39 dependents). The plan sequences contract migration and tests accordingly.

## Desired End State

Consumers still call the orchestrator the same way, but a strategy success is only reported if the outcome is acceptable:

```go
err := orchestrator.Run(ctx, "https://doc.rust-lang.org/", app.OrchestratorOptions{
    Config: cfg,
    FilterURL: "https://doc.rust-lang.org/book/",
})
// Phase 1 result: non-nil error explaining that sitemap produced zero attempted URLs
// and suggesting crawler/filter alternatives. No automatic fallback yet.
```

Strategies now return structured outcomes:

```go
result, err := strategy.Execute(ctx, url, strategyOpts)
verdict := recovery.NewValidator(nil).Validate(result, err, recovery.ValidationOptions{
    FilterURL: opts.FilterURL,
    DryRun:    opts.DryRun,
    MinDocs:   opts.MinDocs,

```

## What We're NOT Doing

- Not building fallback attempts or a plan loop yet.
- Not adding `--strategy`, `--no-fallback`, `--min-docs`, `--explain`, or `--ignore-recipes` CLI flags in this plan.
- Not adding probes (`URLAlive`, `HasOwnSitemap`, etc.).
- Not adding Badger recipe cache or `recipe:` keys.
- Not rewriting the strategy architecture into a discover/filter/fetch/convert/write pipeline.
- Not changing source detection order in `internal/app/detector.go`.
- Not making fallback behavior automatic for manifests yet.

## Decisions

### Scope: Phase 0+1 only

Ambiguity: The research artifact describes Phases 0-5, but its next steps recommend approving Phase 0+1 first. GitNexus reports CRITICAL blast radius for the strategy interface and orchestrator run loop.

Explored:
- Option A: Phase 0+1 only — lower risk and aligns with `SELF-HEALING-PIPELINE.md` next steps.
- Option B: MVP 0-2 — resolves rust-book automatically but rewrites the attempt loop now.
- Option C: Full 0-5 — complete roadmap but a very large plan with probes/cache/UX.

Decision: Build Phase 0+1 only. User selected Phase 0+1.

### Interface migration updates both strategy contracts

Ambiguity: `internal/strategies/strategy.go:25-31` is the active orchestrator contract, while `internal/domain/interfaces.go:10-17` feeds generated mocks in `tests/mocks/domain.go:1-8`.

Explored:
- Option A: Update both — keeps contracts and mocks aligned.
- Option B: Update only `internal/strategies` — less immediate churn, but leaves stale domain mocks.
- Option C: Update only necessary now and unify later — defers a known duplicate contract.

Decision: Update both interfaces in Phase 1. User selected “Atualizar ambas”.

### Skipped documents count toward acceptable output

Ambiguity: The validator could require physical writes only, but existing behavior treats already-present or unchanged documents as successful no-op work.

Explored:
- Option A: Accept `DocsWritten + DocsSkipped >= MinDocs` — preserves repeated runs and sync.
- Option B: Require `DocsWritten` only — stricter but regresses no-change runs.
- Option C: Defer detailed semantics — not enough for Phase 1 validator.

Decision: Skips count. User selected “Skips contam”.

### Dry-run validates attempted extraction, not writes

Simple decision: `--dry-run` means “Simulate without writing files” at `cmd/repodocs/main.go:84`. Requiring `DocsWritten > 0` in dry-run would be a direct regression, so `ValidationOptions.DryRun` treats non-zero attempts without failures as acceptable.

### No fallback in Phase 1

Simple decision: Phase 1 returns a user-facing outcome error instead of executing an alternative. The fallback plan, probes, and CLI flags are explicitly deferred.

### Preserve strategy factory injection

Simple decision: `OrchestratorOptions.StrategyFactory` exists at `internal/app/orchestrator.go:35` and is initialized at `internal/app/orchestrator.go:106-119`; tests should continue to inject fake strategies and now fake `StrategyResult` values.

package domain

import (

// StrategyResult reports the observable outcome of one Strategy.Execute call.
// Counters are populated even when Execute returns an error because partial
// progress is useful for validation and user-facing diagnostics.
type StrategyResult struct {
	mu        sync.Mutex
	startedAt time.Time

	Strategy       string
	EntryURL       string
	URLsDiscovered int
	URLsAttempted  int
	DocsWritten    int
	DocsSkipped    int
	DocsFailed     int
	BytesWritten   int64
	Diagnostics    []Diagnostic
	Duration       time.Duration
}

// Diagnostic is a structured signal emitted by a strategy for the recovery
// validator and later fallback planner phases.
type Diagnostic struct {
	Code    DiagnosticCode
	Message string
	Hint    string
}

// DiagnosticCode identifies a machine-readable extraction outcome signal.
type DiagnosticCode string

const (
	DiagFilterZeroed      DiagnosticCode = "filter_zeroed"
	DiagSitemapShallow    DiagnosticCode = "sitemap_shallow"
	DiagAllFetchesFailed  DiagnosticCode = "all_fetches_failed"
	DiagAllFetchesBlocked DiagnosticCode = "all_fetches_blocked"
	DiagEmptyContent      DiagnosticCode = "empty_content"
	DiagRedirectLoop      DiagnosticCode = "redirect_loop"
	DiagJSRequired        DiagnosticCode = "js_required"
	DiagNoDocuments       DiagnosticCode = "no_documents"
)

// NewStrategyResult creates an empty result for strategy execution.
func NewStrategyResult(strategy, entryURL string) *StrategyResult {
	return &StrategyResult{
		Strategy:  strategy,
		EntryURL:  entryURL,
		startedAt: time.Now(),
	}
}

// NewBasicResult creates a result suitable for mechanical migrations before a
// strategy has detailed counters. Later phases should populate exact counters.
func NewBasicResult(strategy, entryURL string) *StrategyResult {
	return NewStrategyResult(strategy, entryURL)
}

// Finish records elapsed duration once. It is safe to call multiple times.
func (r *StrategyResult) Finish() {
	if r == nil {
		return
	}
	r.mu.Lock()
	defer r.mu.Unlock()
	if r.Duration == 0 && !r.startedAt.IsZero() {
		r.Duration = time.Since(r.startedAt)
	}
}

func (r *StrategyResult) AddDiscovered(n int) {
	if r == nil || n <= 0 {
		return
	}
	r.mu.Lock()
	r.URLsDiscovered += n
	r.mu.Unlock()
}

func (r *StrategyResult) AddAttempted(n int) {
	if r == nil || n <= 0 {
		return
	}
	r.mu.Lock()
	r.URLsAttempted += n
	r.mu.Unlock()
}

func (r *StrategyResult) IncDiscovered() { r.AddDiscovered(1) }
func (r *StrategyResult) IncAttempted()  { r.AddAttempted(1) }

func (r *StrategyResult) IncWritten() {
	if r == nil {
		return
	}
	r.mu.Lock()
	r.DocsWritten++
	r.mu.Unlock()
}

func (r *StrategyResult) IncSkipped() {
	if r == nil {
		return
	}
	r.mu.Lock()
	r.DocsSkipped++
	r.mu.Unlock()
}

func (r *StrategyResult) IncFailed() {
	if r == nil {
		return
	}
	r.mu.Lock()
	r.DocsFailed++
	r.mu.Unlock()
}

func (r *StrategyResult) AddBytesWritten(n int64) {
	if r == nil || n <= 0 {
		return
	}
	r.mu.Lock()
	r.BytesWritten += n
	r.mu.Unlock()
}

func (r *StrategyResult) AddDiagnostic(code DiagnosticCode, message, hint string) {
	if r == nil {
		return
	}
	r.mu.Lock()
	r.Diagnostics = append(r.Diagnostics, Diagnostic{
		Code:    code,
		Message: message,
		Hint:    hint,
	})
	r.mu.Unlock()
}

func (r *StrategyResult) HasDiagnostic(code DiagnosticCode) bool {
	if r == nil {
		return false
	}
	r.mu.Lock()
	defer r.mu.Unlock()
	for _, diagnostic := range r.Diagnostics {
		if diagnostic.Code == code {
			return true
		}
	}
	return false
}

// CompletedDocs returns documents that are acceptable output for validation:
// freshly written files plus skipped existing/synced files.
func (r *StrategyResult) CompletedDocs() int {
	if r == nil {
		return 0
	}
	r.mu.Lock()
	defer r.mu.Unlock()
	return r.DocsWritten + r.DocsSkipped
}

// Snapshot returns a copy without sharing the internal mutex.
func (r *StrategyResult) Snapshot() StrategyResult {
	if r == nil {
		return StrategyResult{}
	}
	r.mu.Lock()
	defer r.mu.Unlock()
	return StrategyResult{
		Strategy:       r.Strategy,
		EntryURL:       r.EntryURL,
		URLsDiscovered: r.URLsDiscovered,
		URLsAttempted:  r.URLsAttempted,
		DocsWritten:    r.DocsWritten,
		DocsSkipped:    r.DocsSkipped,
		DocsFailed:     r.DocsFailed,
		BytesWritten:   r.BytesWritten,
		Diagnostics:    append([]Diagnostic(nil), r.Diagnostics...),
		Duration:       r.Duration,
	}
}

func (s *SitemapStrategy) execute(ctx context.Context, url string, opts Options, result *domain.StrategyResult) error {
	s.logger.Info().Str("url", url).Msg("Fetching sitemap")

	resp, err := s.fetcher.Get(ctx, url)
	if err != nil {
		result.IncFailed()
		return err
	}

	content := resp.Body
	if strings.HasSuffix(strings.ToLower(url), ".gz") {
		content, err = decompressGzip(resp.Body)
		if err != nil {
			result.IncFailed()
			return err
		}
	}

	sitemap, err := parseSitemap(content, url)
	if err != nil {
		result.IncFailed()
		return err
	}

	if sitemap.IsIndex {
		return s.processSitemapIndex(ctx, sitemap, opts, result)
	}

	sortURLsByLastMod(sitemap.URLs)
	result.AddDiscovered(len(sitemap.URLs))

	urls := sitemap.URLs
	if opts.Limit > 0 && len(urls) > opts.Limit {
		urls = urls[:opts.Limit]
	}

	if opts.FilterURL != "" {
		beforeFilter := len(urls)
		urls = filterSitemapURLs(urls, opts.FilterURL)
		s.logger.Info().Str("filter", opts.FilterURL).Int("filtered_count", len(urls)).Msg("URLs after filter")
		if beforeFilter > 0 && len(urls) == 0 {
			result.AddDiagnostic(domain.DiagFilterZeroed,
				"URL filter excluded every sitemap URL",
				"Use the filtered URL as the crawler entry point or adjust --filter")
		}
		if beforeFilter > 0 && beforeFilter < 10 {
			result.AddDiagnostic(domain.DiagSitemapShallow,
				"Sitemap contains very few URLs before filtering",
				"A shallow sitemap is a weak signal; crawler may be a better strategy")
		}
	}

	s.logger.Info().Int("count", len(urls)).Msg("Processing URLs from sitemap")
	return s.processURLs(ctx, urls, opts, result)
}

func (s *SitemapStrategy) processSitemapIndex(ctx context.Context, sitemap *domain.Sitemap, opts Options, result *domain.StrategyResult) error {
	s.logger.Info().Int("count", len(sitemap.Sitemaps)).Msg("Processing sitemap index")

	if opts.FilterURL != "" {
		s.logger.Info().Str("filter", opts.FilterURL).Msg("URL filter active - only processing URLs under this path")
	}

	totalProcessed := 0
	totalDiscovered := 0
	for _, sitemapURL := range sitemap.Sitemaps {
		select {
		case <-ctx.Done():
			return ctx.Err()
		default:
		}

		urls, discovered, err := s.collectURLsFromSitemap(ctx, sitemapURL, opts.FilterURL)
		totalDiscovered += discovered
		result.AddDiscovered(discovered)
		if err != nil {
			result.IncFailed()
			s.logger.Warn().Err(err).Str("url", sitemapURL).Msg("Failed to fetch nested sitemap")
			continue
		}

		if len(urls) == 0 {
			continue
		}

		if opts.Limit > 0 {
			remaining := opts.Limit - totalProcessed
			if remaining <= 0 {
				break
			}
			if len(urls) > remaining {
				urls = urls[:remaining]
			}
		}

		s.logger.Info().Int("count", len(urls)).Str("sitemap", sitemapURL).Msg("Processing URLs from nested sitemap")
		if err := s.processURLs(ctx, urls, opts, result); err != nil {
			return err

func (s *SitemapStrategy) collectURLsFromSitemap(ctx context.Context, url string, filterURL string) ([]domain.SitemapURL, int, error) {
	resp, err := s.fetcher.Get(ctx, url)
	if err != nil {
		return nil, 0, err
	}

	content := resp.Body
	if strings.HasSuffix(strings.ToLower(url), ".gz") {
		content, err = decompressGzip(resp.Body)
		if err != nil {
			return nil, 0, err
		}
	}

	sitemap, err := parseSitemap(content, url)
	if err != nil {
		return nil, 0, err
	}

	if sitemap.IsIndex {
		var allURLs []domain.SitemapURL
		var discovered int
		for _, nestedURL := range sitemap.Sitemaps {
			select {
			case <-ctx.Done():
				return nil, discovered, ctx.Err()
			default:
			}

			urls, nestedDiscovered, err := s.collectURLsFromSitemap(ctx, nestedURL, filterURL)
			discovered += nestedDiscovered
			if err != nil {
				s.logger.Warn().Err(err).Str("url", nestedURL).Msg("Failed to fetch nested sitemap")
				continue
			}
			allURLs = append(allURLs, urls...)
		}
		return allURLs, discovered, nil
	}

	discovered := len(sitemap.URLs)
	if filterURL != "" {
		sitemap.URLs = filterSitemapURLs(sitemap.URLs, filterURL)
	}
	return sitemap.URLs, discovered, nil
}

				return nil
			}
		} else {

		}

func (s *CrawlerStrategy) execute(ctx context.Context, url string, opts Options, result *domain.StrategyResult) error {
	s.logger.Info().Str("url", url).Msg("Starting web crawl")

	if opts.FilterURL != "" {
		s.logger.Info().Str("filter", opts.FilterURL).Msg("URL filter active - only crawling URLs under this path")
	}

	result.IncDiscovered()
	cctx := newCrawlContext(ctx, url, opts, result)

	c := colly.NewCollector(
		colly.Async(true),
		colly.MaxDepth(opts.MaxDepth),
	)
	cctx.collector = c

	if fetcherClient, ok := s.fetcher.(*fetcher.Client); ok {
		c.WithTransport(fetcherClient.TransportWithOptions(fetcher.StealthTransportOptions{
			RendererFallback: s.makeRendererFallback(),
			Logger:           s.logger,
		}))
	} else {
		c.WithTransport(s.fetcher.Transport())
	}

	_ = c.Limit(&colly.LimitRule{
		DomainGlob:  "*",
		Parallelism: opts.Concurrency,
		RandomDelay: 2 * time.Second,
	})

	c.OnHTML("a[href]", func(e *colly.HTMLElement) {
		link := e.Request.AbsoluteURL(e.Attr("href"))
		if s.shouldProcessURL(link, url, cctx) {
			_ = e.Request.Visit(link)
		}
	})

	c.OnResponse(func(r *colly.Response) {
		s.processResponse(ctx, r, cctx)
	})

	c.OnError(func(r *colly.Response, err error) {
		result.IncFailed()
		s.logger.Debug().Err(err).Str("url", r.Request.URL.String()).Msg("Request failed")
	})

	if err := c.Visit(url); err != nil {
		result.IncFailed()
		return err
	}

	done := make(chan struct{})
	go func() {
		c.Wait()
		close(done)
	}()

	select {
	case <-ctx.Done():
		return ctx.Err()
	case <-done:
	}

	snapshot := result.Snapshot()
	if snapshot.URLsAttempted == 0 && snapshot.DocsFailed > 0 {
		result.AddDiagnostic(domain.DiagAllFetchesFailed,
			"Crawler visited URLs but no HTML or Markdown responses were processed",
			"Check robots, status codes, content type, or try a more specific entry URL")
	}

	s.logger.Info().Int("pages", *cctx.processedCount).Msg("Crawl completed")
	return nil
}

			result.IncSkipped()
			return nil

		if err != nil {

			result.IncFailed()
			return nil
		}
		if len(strings.TrimSpace(doc.Content)) < 50 {
			result.IncSkipped()
			return nil

				return nil

func (p *Processor) ProcessFiles(ctx context.Context, files []string, tmpDir string, opts ProcessOptions) error {
	if opts.Result != nil {
		opts.Result.AddAttempted(len(files))
	}
	bar := utils.NewProgressBar(len(files), utils.DescExtracting)
	errors := utils.ParallelForEach(ctx, files, opts.Concurrency, func(ctx context.Context, file string) error {
		defer bar.Add(1)
		if err := p.ProcessFile(ctx, file, tmpDir, opts); err != nil {
			if opts.Result != nil {
				opts.Result.IncFailed()
			}
			if p.logger != nil {
				p.logger.Warn().Err(err).Str("file", file).Msg("Failed to process file")
			}
		}
		return nil
	})
	return utils.FirstError(errors)
}

func (p *Processor) ProcessFile(ctx context.Context, path, tmpDir string, opts ProcessOptions) error {
	info, err := os.Stat(path)
	if err != nil {
		return err
	}
	maxSize := opts.MaxFileSize
	if maxSize == 0 {
		maxSize = 10 * 1024 * 1024
	}
	if maxSize > 0 && info.Size() > maxSize {
		if opts.Result != nil {
			opts.Result.IncSkipped()
		}
		return nil
	}

	content, err := os.ReadFile(path)
	if err != nil {
		return err
	}
	// keep document construction/conversion unchanged

	if opts.StateManager != nil {
		opts.StateManager.MarkSeen(fileURL)
		if !opts.StateManager.ShouldProcess(fileURL, contentHash) {
			if opts.Result != nil {
				opts.Result.IncSkipped()
			}
			return nil
		}
	}

	if !opts.DryRun && opts.WriteFunc != nil {
		if err := opts.WriteFunc(ctx, doc); err != nil {
			return err
		}
		if opts.Result != nil {
			opts.Result.IncWritten()
			opts.Result.AddBytesWritten(info.Size())
		}
	}

package recovery

import (
	"fmt"

	"github.com/quantmind-br/repodocs/internal/domain"
)

const (
	DefaultMinDocsWritten  = 1
	DefaultMinSuccessRatio = 0.10
)

type Verdict interface{ verdict() }

type VerdictOK struct{}

type VerdictRetryAlternative struct {
	Reason      string
	Diagnostics []domain.Diagnostic
}

type VerdictHardFail struct {
	Reason string
	Cause  error
}

type VerdictPropagate struct {
	Cause error
}

func (VerdictOK) verdict()               {}
func (VerdictRetryAlternative) verdict() {}
func (VerdictHardFail) verdict()         {}
func (VerdictPropagate) verdict()        {}

type Criteria struct {
	MinDocsWritten int
	MinSuccessRatio float64
}

type ValidationOptions struct {
	FilterURL       string
	DryRun          bool
	MinDocs         int
	MinSuccessRatio float64
}

type Validator struct {
	criteria Criteria
}

func NewValidator(criteria *Criteria) *Validator {
	resolved := Criteria{
		MinDocsWritten: DefaultMinDocsWritten,
		MinSuccessRatio: DefaultMinSuccessRatio,
	}
	if criteria != nil {
		if criteria.MinDocsWritten > 0 {
			resolved.MinDocsWritten = criteria.MinDocsWritten
		}
		if criteria.MinSuccessRatio > 0 {
			resolved.MinSuccessRatio = criteria.MinSuccessRatio
		}
	}
	return &Validator{criteria: resolved}
}

func (v *Validator) Validate(r *domain.StrategyResult, err error, opts ValidationOptions) Verdict {
	criteria := v.criteriaFor(opts)

	if err != nil && domain.IsTransient(err) {
		return VerdictPropagate{Cause: err}
	}
	if err != nil {
		return VerdictHardFail{Reason: "strategy execution failed", Cause: err}
	}
	if r == nil {
		return VerdictHardFail{Reason: "strategy returned no outcome telemetry", Cause: domain.ErrInsufficientOutput}
	}

	r.Finish()
	snapshot := r.Snapshot()
	completedDocs := snapshot.DocsWritten + snapshot.DocsSkipped

	if completedDocs >= criteria.MinDocsWritten {
		return VerdictOK{}
	}
	if opts.DryRun && snapshot.URLsAttempted >= criteria.MinDocsWritten && snapshot.DocsFailed == 0 {
		return VerdictOK{}
	}
	if snapshot.URLsAttempted == 0 && opts.FilterURL != "" {
		return VerdictRetryAlternative{
			Reason:      string(domain.DiagFilterZeroed),
			Diagnostics: diagnosticsOrDefault(snapshot, domain.DiagFilterZeroed, "URL filter excluded all candidate URLs"),
		}
	}
	if snapshot.URLsDiscovered > 0 && snapshot.URLsAttempted == 0 {
		return VerdictRetryAlternative{
			Reason:      "no_urls_attempted",
			Diagnostics: diagnosticsOrDefault(snapshot, domain.DiagNoDocuments, "No discovered URLs survived filtering or limits"),
		}
	}
	if snapshot.URLsAttempted > 20 {
		ratio := float64(completedDocs) / float64(snapshot.URLsAttempted)
		if ratio < criteria.MinSuccessRatio {
			return VerdictRetryAlternative{
				Reason:      fmt.Sprintf("high_failure_ratio: %.2f", ratio),
				Diagnostics: snapshot.Diagnostics,
			}
		}
	}
	if completedDocs == 0 {
		return VerdictHardFail{Reason: "extraction produced 0 documents", Cause: domain.ErrInsufficientOutput}
	}
	return VerdictHardFail{Reason: "extraction below minimum output threshold", Cause: domain.ErrInsufficientOutput}
}

func (v *Validator) criteriaFor(opts ValidationOptions) Criteria {
	criteria := v.criteria
	if opts.MinDocs > 0 {
		criteria.MinDocsWritten = opts.MinDocs
	}
	if opts.MinSuccessRatio > 0 {
		criteria.MinSuccessRatio = opts.MinSuccessRatio
	}
	return criteria
}

func diagnosticsOrDefault(snapshot domain.StrategyResult, code domain.DiagnosticCode, message string) []domain.Diagnostic {
	if len(snapshot.Diagnostics) > 0 {
		return snapshot.Diagnostics
	}
	return []domain.Diagnostic{{Code: code, Message: message}}
}

package recovery

import (
	"fmt"
	"strings"

	"github.com/quantmind-br/repodocs/internal/domain"
)

type OutcomeError struct {
	Verdict     Verdict
	Result      *domain.StrategyResult
	Suggestions []string
}

func NewOutcomeError(verdict Verdict, result *domain.StrategyResult) *OutcomeError {
	return &OutcomeError{
		Verdict:     verdict,
		Result:      result,
		Suggestions: suggestionsFor(verdict, result),
	}
}

func (e *OutcomeError) Error() string {
	var b strings.Builder
	reason := verdictReason(e.Verdict)
	if reason == "" {
		reason = "extraction outcome was unsatisfactory"
	}
	fmt.Fprintf(&b, "extraction outcome unsatisfactory: %s", reason)

	if e.Result != nil {
		snapshot := e.Result.Snapshot()
		fmt.Fprintf(&b, "\n\nStrategy:    %s", snapshot.Strategy)
		fmt.Fprintf(&b, "\nEntry URL:   %s", snapshot.EntryURL)
		fmt.Fprintf(&b, "\nDiscovered:  %d URLs", snapshot.URLsDiscovered)
		fmt.Fprintf(&b, "\nAttempted:   %d URLs", snapshot.URLsAttempted)
		fmt.Fprintf(&b, "\nWritten:     %d docs", snapshot.DocsWritten)
		fmt.Fprintf(&b, "\nSkipped:     %d docs", snapshot.DocsSkipped)
		fmt.Fprintf(&b, "\nFailed:      %d docs", snapshot.DocsFailed)
		if len(snapshot.Diagnostics) > 0 {
			b.WriteString("\nDiagnostics:")
			for _, diagnostic := range snapshot.Diagnostics {
				fmt.Fprintf(&b, "\n  - %s: %s", diagnostic.Code, diagnostic.Message)
				if diagnostic.Hint != "" {
					fmt.Fprintf(&b, " (%s)", diagnostic.Hint)
				}
			}
		}
	}

	if len(e.Suggestions) > 0 {
		b.WriteString("\n\nSuggestions:")
		for _, suggestion := range e.Suggestions {
			fmt.Fprintf(&b, "\n  - %s", suggestion)
		}
	}
	return b.String()
}

func (e *OutcomeError) Unwrap() error {
	switch verdict := e.Verdict.(type) {
	case VerdictHardFail:
		return verdict.Cause
	case VerdictPropagate:
		return verdict.Cause
	default:
		return nil
	}
}

func verdictReason(verdict Verdict) string {
	switch v := verdict.(type) {
	case VerdictRetryAlternative:
		return v.Reason
	case VerdictHardFail:
		return v.Reason
	case VerdictPropagate:
		if v.Cause != nil {
			return v.Cause.Error()
		}
	}
	return ""
}

func suggestionsFor(verdict Verdict, result *domain.StrategyResult) []string {
	var suggestions []string
	if retry, ok := verdict.(VerdictRetryAlternative); ok {
		switch retry.Reason {
		case string(domain.DiagFilterZeroed), "no_urls_attempted":
			suggestions = append(suggestions,
				"Try the filtered URL as the entry point instead of only --filter",
				"Check filter URL spelling and trailing slash",
				"Force a different strategy after Phase 2 adds --strategy")
		}
	}
	if result != nil {
		snapshot := result.Snapshot()
		if snapshot.Strategy == "sitemap" && snapshot.URLsDiscovered > 0 && snapshot.URLsAttempted == 0 {
			suggestions = append(suggestions, "The sitemap may be too shallow for this path; try crawler on the filtered URL")
		}
	}
	if len(suggestions) == 0 {
		suggestions = append(suggestions, "Run again with --verbose to inspect strategy diagnostics")
	}
	return suggestions
}
